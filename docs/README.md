## Local preview

```bash
npm install
npx serve docs -p 3338
# open http://localhost:3338/
```

To preview the full hub **with** a local report:

```bash
npm run test:report
mkdir -p docs/report
cp -R results/report/* docs/report/
npx serve docs -p 3338
```
